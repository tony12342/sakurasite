Lista de comandos » Economia

  1. Atm / Bal / Balance
     » Consulta o saldo do autor do comando ou o saldo do usuário mencionado.
    Uso: sa!atm [@user]
  2. Bet / Apostar / Aposta
     » Aposta moedas com outros usuários.
    Uso: sa!bet <@user> <quantia>
  3. Daily / Diário
    » Coleta moedas e pode ser usado uma vez a cada 24 horas.
    Uso: sa!daily
  4. Guessthenumber / Gtn / Advnumber
    » Aposta moedas, o objetivo é você conseguir adivinhar qual o número correto, caso você consiga você ganha o valor apostado.
    Uso: sa!gtn <valor> <Número de 1 a 10>
  5. Pay / Pagar
    » Envia moedas para o usuário mencionado.
    Uso: sa!pay <@user> <quantia>
  6. Rifa / Raffle
    » Mostra informações sobre a rifa. Utilize sa!rifa buy <quantia> para comprar tickets para a rifa.
    Uso: sa!rifa
  7. Rank / Top
    » Informa o rank global de moedas.
    Uso: sa!rank [1/2/3/...]
  8. Rinha / Race
    » Inicia uma rinha, onde vários users podem participar, você escolhe o valor de entrada e o limite de usuários. O prêmio vai acumulando de acordo com os usuários que entram na rinha e apenas um leva o prêmio!
    Uso: sa!rinha <quantia> [limite de usuários de 1 a 20]
  9. Roleta / Roul / Roulette
    » Aposta moedas em uma cor, você pode escolher entre três cores, use sa!roleta para saber mais.
    Uso: sa!roleta <aposta> <cor>
  10. Slots / Slot / Sl
    » Joga no slot valendo moedas, podendo perder as moedas apostadas ou ganhar caso os três emojis caiam iguais.
    Uso: sa!slot <quantia>
  11. Tr / T / Transações / Trs / Transactions
    » Mostra as transações do usuário mencionado e caso não seja mencionado ninguém, mostra as transações de quem executou o comando.
    Uso: sa!transações [@user]
  12. Work / Trabalhar / Trabalho / Job
    » Você trabalha, coleta moedas e ganha XP, quanto maior seu nível mais moedas você ganha trabalhando, utilize sa!work info para ter algumas informaçõese e sa!work name <nome> para mudar o nome da sua empresa.
    Uso: sa!work [info/name]

    Legenda:
    < >: Obrigatório
    [ ]: Opcional
    A barra divide sinônimos de nomes de algum comando.

Lista de comandos » Moderação

  1. Cmdblock / Cmd-block / Cmdb / Cblock / Commandblock
    » Faz com que a Sakura ignore comandos solicitados em canais especificos que você pode escolher quais são. Use sa!cmdblock info para ter algumas informações, sa!cmdblock addchannel <canal> bloqueia um canal de utilizar os comandos, sa!cmdblock addcmd <comando> libera um comando para ser utilizado nos canais, sa!cmdblock msg <mensagem>, configura a mensagem que a Sakura envia quando o usuário usa algum comando em um canal não permitido, algumas ferramentas que você pode utilizar na mensagem da bot: 
    {author} - Menciona o usuário.
    {author.tag} - Mostra a tag do usuário.
    Uso: sa!cmdblock

Lista de comandos » Informações

    » Mostra algumas informações sobre a Sakura.
    Uso: sa!help
  2. Ping

    Uso: sa!ping
